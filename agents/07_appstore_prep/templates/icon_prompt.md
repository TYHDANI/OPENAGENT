# App Icon Generation Prompt Template

Use this template to generate an app icon with Nano Banana Pro or similar AI image generator.

## Prompt Structure

```
A modern iOS app icon for "{{APP_NAME}}", a {{APP_CATEGORY}} app.

Style: {{STYLE — e.g., flat design, gradient, 3D render, minimalist}}
Primary colors: {{2-3 colors that match the app's brand}}
Symbol: {{Main symbol/icon — should be simple and recognizable at small sizes}}
Background: {{Solid color, gradient, or subtle pattern}}
Mood: {{Professional, playful, calm, energetic, etc.}}

Requirements:
- 1024x1024 pixels, PNG format
- No text or letters in the icon
- Simple, recognizable at 29x29 points
- Follows Apple Human Interface Guidelines
- No transparency
- Square with no rounded corners (iOS applies mask automatically)
```

## Apple HIG Guidelines for Icons
- Use a simple, recognizable shape
- Use a single focal point
- Keep backgrounds simple
- Don't include text (app name shows below)
- Use 2-3 colors maximum
- Avoid photos — use illustrations or symbols
- Test at small sizes (29pt, 40pt, 60pt)
