# App Preview Video Generation — Votion Prompt Template

Use this template to generate an App Store preview video with Votion or similar AI video tools.

## Video Specifications (Apple Requirements)
- **Duration**: 15-30 seconds
- **Resolution**: 1080x1920 (9:16 portrait for iPhone)
- **FPS**: 30
- **Format**: H.264, .mp4

## Scene Breakdown Template

### Scene 1: Hook (0-3s)
```
Show the app's main value proposition in a single powerful visual.
Text overlay: "{{ONE_LINE_VALUE_PROP}}"
Background: {{App's primary brand color}} with subtle animation
Transition: Smooth fade into Scene 2
```

### Scene 2: Problem (3-7s)
```
Illustrate the problem the app solves.
Text overlay: "{{PROBLEM_STATEMENT — e.g., 'Tired of forgetting important tasks?'}}"
Visual: {{Relatable scenario showing frustration/inefficiency}}
Transition: Swipe left into Scene 3
```

### Scene 3: Solution Demo (7-17s)
```
Show the app in action — this is the hero section.
Record key workflow: {{MAIN_USER_FLOW — e.g., 'Create task → Set reminder → Mark complete'}}
Highlight 2-3 key features with callout annotations.
Show real app screens (from screenshots).
Transition: Zoom into Scene 4
```

### Scene 4: Social Proof / Results (17-22s)
```
Show the outcome / benefit.
Text overlay: "{{RESULT — e.g., 'Never miss a deadline again'}}"
Visual: {{Satisfied user scenario or impressive stats}}
Transition: Fade into Scene 5
```

### Scene 5: CTA (22-30s)
```
Final call to action.
Text overlay: "Try {{APP_NAME}} Free"
Show: App icon + "Download on the App Store" badge
Background: {{Brand gradient}}
```

## Audio
- Background music: Upbeat, modern, royalty-free
- No voiceover (Apple allows but not required)
- Sound effects: Subtle UI interaction sounds

## Notes
- No pricing information in the video (Apple policy)
- Show only real app functionality
- Include accessibility features if applicable
